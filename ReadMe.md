# Clone 후 세팅

### node.js (npm), git 이 설치되었다는 가정하에 아래의 내용을 수행할것.

#### 아래 명령어는 git bash에서 수행

`$ npm install -g hexo-cli`

그런 다음 클론한 폴더내에서

`$ hexo init <폴더명>`

이후 로컬에서 서버를 띄워본다.

`$ hexo s ( or server )`

서버가 정상적으로 띄워지지않거나, 오류가 발생하여 수정하였다면 아래와 같이.

`$ hexo clean `

<http://localhost:4000> 로 접속해서 정상적으로 되면 hexo 성공

추가내용이 생기면 추가 작성예정
